# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-18

### Added
- Initial release of TempMailChecker PHP SDK
- `isDisposable()` method for quick boolean checks
- `check()` method for full email validation
- `isDisposableDomain()` method for domain-only checks
- `checkDomain()` method for domain validation
- `getUsage()` method to check API usage statistics
- Support for regional endpoints (US, EU, Asia)
- Comprehensive error handling
- Full PSR-4 autoloading support
- Composer package configuration
